# WaterLevelPredction_SelectingModel

Bài toán đặt ra: dữ liệu vận hành của 1 hồ chứa nông nghiệp, từ các dữ liệu này tạo ra 1 mô hình học máy có thể dự doán mực nước trong hồ dựa vào các thông số đầu vào

Project bao gồm

Tiền xử lý dữ liệu

Sử dung 3 thuật toán để huấn luyện mô hình học máy
- Linear Regression
- Random Forest
- XGBoost

Sử dụng các chỉ số như rmse, mae, r2 score để so sánh và lựa chọn ra mô hình tốt nhất cho bài toán

Đóng gói model và deploy
